Environment: server.  
Default value: `'en'`.

To set the [`lang` attribute](https://developer.mozilla.org/en-US/docs/Web/HTML/Global_attributes/lang#language_tag_syntax):

```js
// /pages/+config.h.js

export default {
  lang: 'fr'
}
```
