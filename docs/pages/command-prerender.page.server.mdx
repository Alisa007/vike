Vike CLI command to trigger <Link href="/pre-rendering">pre-rendering</Link>.

```bash
$ vike prerender
```

> Use <Link href="/prerender-config#disableautorun" /> if you don't want pre-rendering to be run twice.

The Vike CLI doesn't accept any option, use the config file instead.
